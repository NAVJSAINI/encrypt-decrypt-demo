Encrypt/Decrypt Demo

This project demonstrates symmetric and asymmetric encryption using Python.

Symmetric Encryption:
Uses a single key to encrypt and decrypt data. It is fast but requires secure key sharing.

Asymmetric Encryption:
Uses a public key to encrypt and a private key to decrypt. It is more secure but slower.

Login System:
A simple login system checks username and password and gives access based on role.

Files Included:
- encryption_demo.ipynb
- README.txt

Conclusion:
Symmetric encryption is efficient but less secure for sharing keys.
Asymmetric encryption is more secure but slower.
